 Análisis de Elasticidad de la Demanda y Estrategia de Precios

 Descripción

Este proyecto analiza el comportamiento de la demanda frente a cambios en el precio de distintos productos, utilizando SQL para identificar niveles de elasticidad y optimizar estrategias de precios.

 Herramientas utilizadas

* SQL (PostgreSQL)
* pgAdmin

 Análisis realizado

Se evaluó la relación entre precio y cantidad demandada para diferentes productos a lo largo del tiempo, identificando patrones de comportamiento en la demanda.

 Pantalón

Se identificó como un producto altamente elástico, ya que incrementos en el precio generan caídas significativas en la cantidad demandada.

 Playera

Presenta un comportamiento intermedio, donde existe un punto óptimo de precio que maximiza la ganancia, equilibrando volumen de ventas y margen.

 Zapatos

Se comporta como un producto inelástico, ya que la demanda no disminuye significativamente ante aumentos de precio.

 Hallazgos clave

* Existen diferentes niveles de sensibilidad al precio entre productos
* No todos los productos deben seguir la misma estrategia de precios
* Identificar elasticidad permite maximizar ingresos y rentabilidad

 Recomendaciones

* Mantener precios competitivos en productos elásticos (pantalón)
* Optimizar precios en productos intermedios (playera)
* Evaluar incrementos de precio en productos inelásticos (zapatos)
 
 Estructura del proyecto

* `queries.sql`: consultas SQL utilizadas
* `README.md`: explicación del análisis
 Conclusión

El análisis de elasticidad permite tomar decisiones estratégicas basadas en datos, mejorando la rentabilidad del negocio y optimizando la relación entre precio y demanda.
